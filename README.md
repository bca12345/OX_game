## OX game using TCP/IP socket programming in C

'O': client
'X': server
type the number to decide the place you put 'O' or 'X'

### Usage

1.Create server
  
  
    ./OX_server

2.Run client and connect to localhost
  
  
    ./OX_client 127.0.0.1
    
3. It will show as the result
